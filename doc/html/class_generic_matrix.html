Detailed Description

template<typename Elem = float>
class GenericMatrix< Elem >

The GenericMatrix class defines a generic template matrix class, the matrix elements are managed by a one-dimension pointer.

The GenericMatrix template has one parameter:

  • Elem Element type that is visible to users of the class.
