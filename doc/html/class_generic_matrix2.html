Detailed Description

template<typename Elem = float>
class GenericMatrix2< Elem >

The GenericMatrix2 class defines a generic template matrix class, the matrix elements are managed by a two-dimension pointer.

The GenericMatrix2 template has one parameter:

  • Elem Element type that is visible to users of the class.
